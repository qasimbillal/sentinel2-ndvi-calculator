# Sentinel-2 NDVI Calculator

Python tool to download Sentinel-2 imagery and compute NDVI for a polygon.

Features:
- Sentinel-2 STAC API search
- Automatic band download
- Raster clipping
- NDVI calculation
- NDVI raster export
- NDVI point sampling

Output:
- ndvi.tif
- ndvi_points.csv

Run:

python run_ndvi.py
